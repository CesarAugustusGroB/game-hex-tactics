Referencia técnica del juego

Wargame táctico por hexágonos

Aplicación PIXI.js de un solo canvas con un HUD React delgado. Un mismo generador procedural produce la vista estratégica (isla) y la táctica (campo de batalla) — se "bucea" de una a otra. El simulador de batalla es una función pura; el enemigo es una IA por reglas + percepción.

El loop de juego

Tick de 500 ms: regen de CP → IA emite órdenes → simulateTick → scoring → detección de victoria.

Ver el tick →
